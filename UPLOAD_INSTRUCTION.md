# Как загрузить проект в новый репозиторий GitHub Pages

1. Создайте новый репозиторий, например `etagi-raskleyka`.
2. Загрузите содержимое архива в корень репозитория, не в дополнительную вложенную папку.
3. В корне должны лежать:
   - `index.html`
   - `README.md`
   - папки `assets`, `data`, `docs`
4. Откройте Settings → Pages.
5. Выберите:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / root
6. Сохраните настройки.

После публикации сайт будет доступен по адресу:

`https://deputat36.github.io/НАЗВАНИЕ_РЕПОЗИТОРИЯ/`

Важно: не загружайте этот проект в `vktg`, если там должен оставаться другой проект.

# История изменений

## 1.2.0

- Убраны быстрые кнопки районов: район теперь вводится вручную.
- Добавлен блок «Состав макета».
- Можно включать и выключать: заголовок, цену, описание, параметры, преимущества, фото, QR, контакты.
- Рендер макета перестраивается под включенные блоки.
- Проверка качества теперь учитывает скрытые блоки.

## 1.1.0

- Добавлено сохранение профиля СПН отдельно от макета.
- Добавлены кнопки «Сохранить профиль СПН», «Подставить профиль», «Очистить объект».
- Добавлены быстрые типы объектов.
- При выборе нового шаблона имя и телефон СПН больше не сбрасываются.

## 1.0.0

- Первая модульная версия под GitHub Pages.
- Простая работа для СПН.
- Гибкая настройка для опытных пользователей.
- Шаблоны в JSON.
- Поддержка фото 1/2, QR, деления А4, контроля качества.

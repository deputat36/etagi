# История изменений

## 1.7.0

- Добавлен произвольный дополнительный текстовый блок.
- Дополнительный блок можно включать и выключать в составе макета.
- Добавлены поля: название дополнительного блока и текст дополнительного блока.
- Блок подходит для условий, ипотеки, торга, срочности, бонусов и важных уточнений.
- Проверка качества теперь учитывает пустой или слишком длинный дополнительный блок.

## 1.6.0

- Добавлено сохранение нескольких макетов с названиями.
- Добавлен список сохранённых макетов.
- Добавлены действия: сохранить как макет, загрузить выбранный, удалить выбранный.
- Именованные макеты хранятся в браузере отдельно от последнего автосохранения.
- Фото не сохраняются в именованные макеты, чтобы не переполнять localStorage.

## 1.5.0

- Добавлены настройки печати: линии реза, безопасные поля, режим проверки печати.
- Предпросмотр А4 теперь умеет показывать зоны реза и безопасную область внутри каждого макета.
- Проверка качества теперь подсказывает включить линии реза и безопасные поля.
- Окно печати дополнено чек-листом: A4, масштаб 100%, фоновые изображения, проверка QR.

## 1.4.0

- Добавлен настоящий QR-код вместо визуальной заглушки.
- QR генерируется прямо в браузере, без сервера и внешних API.
- Добавлен модуль `assets/js/qr.js`.
- Добавлена проверка слишком длинных ссылок для QR.
- Добавлены стили для печатного QR-кода.

## 1.3.0

- Добавлен модуль `assets/js/layoutRules.js` для умной подстройки макета.
- Добавлены быстрые режимы макета: «Читаемо», «Экономно», «С фото», «Витрина», «Подъезд», «Частное».
- Добавлена кнопка «Подстроить автоматически».
- Добавлены подсказки по составу макета.
- Автоподстройка меняет количество макетов на А4, плотность, фото, QR, отрывные телефоны, размеры заголовка и телефона.

## 1.2.0

- Убраны быстрые кнопки районов: район теперь вводится вручную.
- Добавлен блок «Состав макета».
- Можно включать и выключать: заголовок, цену, описание, параметры, преимущества, фото, QR, контакты.
- Рендер макета перестраивается под включенные блоки.
- Проверка качества теперь учитывает скрытые блоки.

## 1.1.0

- Добавлено сохранение профиля СПН отдельно от макета.
- Добавлены кнопки «Сохранить профиль СПН», «Подставить профиль», «Очистить объект».
- Добавлены быстрые типы объектов.
- При выборе нового шаблона имя и телефон СПН больше не сбрасываются.

## 1.0.0

- Первая модульная версия под GitHub Pages.
- Простая работа для СПН.
- Гибкая настройка для опытных пользователей.
- Шаблоны в JSON.
- Поддержка фото 1/2, QR, деления А4, контроля качества.

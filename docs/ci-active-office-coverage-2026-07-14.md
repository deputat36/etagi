# Контрольный CI активного office-покрытия

Служебный документ для проверки актуального `main`.

Ожидается:

```text
validate:template-portfolio → success
```

Freshness-проверка должна подтвердить:

- все `working` и `test` шаблоны имеют office-метаданные;
- отсутствие office допускается только у `deprecated`;
- generated-инвентаризация остаётся актуальной;
- полный browser-smoke и печатная матрица остаются зелёными.

PR служебный. Полезный код уже находится в `main`.

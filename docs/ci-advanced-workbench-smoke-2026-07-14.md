# Контрольный CI дополнительной аналитики

Служебный документ для проверки актуального `main` после расширения browser-smoke.

Ожидаемый маршрут:

```text
validate → browser-smoke → 5 print-screenshot job → collect-print-screenshots
```

Browser-smoke должен подтвердить:

- контейнер `Дополнительная аналитика` скрыт в режиме `Быстро`;
- в режиме `Расширенно` внутри находятся три исходные карточки;
- введённая заметка сохраняется после закрытия и повторного открытия;
- печатные сценарии не меняются.

PR служебный. Полезный код уже находится в `main`.

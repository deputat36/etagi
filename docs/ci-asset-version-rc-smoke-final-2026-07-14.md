# Финальный контроль asset-version и расширенного smoke

Служебный документ для запуска актуального `main`.

Проверяется:

```text
validate → browser-smoke → print-screenshot × 5 → collect-print-screenshots
```

Критерии:

- entry-assets имеют единый `?v=3.85.0`;
- asset-version и RC-gate проходят статический барьер;
- расширенный smoke получает 65 секунд внутри harness и 85 секунд процесса Chrome;
- отдельные проверки по-прежнему имеют короткие строгие таймауты;
- пять печатных сценариев остаются зелёными.

PR служебный. Полезный код уже находится в `main`.

# Контрольный CI пакета ручной приёмки

Служебный документ для запуска актуального `main`.

Проверяется:

```text
validate → browser-smoke → print-screenshot × 5 → collect-print-screenshots
```

Критерии:

- release-validator требует `manual-print-test-pack-3.86.0.md`;
- пакет содержит единые данные, пять групп сценариев, QR двумя телефонами и критерий расхода чернил;
- RC остаётся `DRAFT` до заполнения основного бланка;
- runtime и печатная матрица остаются зелёными.

PR служебный. Полезный код уже находится в `main`.

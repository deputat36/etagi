# Контрольный CI блока после печати

Служебный документ для проверки актуального `main`.

Ожидаемый маршрут:

```text
validate → browser-smoke → 5 print-screenshot job → collect-print-screenshots
```

Проверяется:

- новый `validate:post-print-workspace`;
- исходно закрытый контейнер;
- автоматическое открытие на шагах `Задание` и `Отчёт`;
- возврат к сохранённому состоянию после выхода из `Новичка`;
- live-статус задания и отчёта;
- сохранность режима исполнителя, истории и аналитики.

PR служебный. Полезный код уже находится в `main`.

# Контрольный CI изолированной screenshot-матрицы

Служебный документ для запуска актуального `main`.

Ожидаемый маршрут:

```text
validate → browser-smoke → 5 print-screenshot matrix jobs → collect-print-screenshots
```

Проверяются пять листов А4, комплектность PNG, общий manifest и оптимизированная контактная зона без большой сплошной тёмной заливки.

PR служебный. Код уже находится в `main`, в diff только этот документ.

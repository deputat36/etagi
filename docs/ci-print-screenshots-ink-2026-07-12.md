# Контрольный CI печатных PNG и расхода чернил

Служебный документ для наблюдаемого pull-request запуска актуального `main`.

Ожидаются три последовательных job:

```text
validate → browser-smoke → print-screenshots
```

Проверяются:

- пять PNG печатных листов А4;
- отсутствие overflow и наложения контактов;
- крупный телефон;
- фото-компоновки;
- облегчённая цветная контактная зона в `four-contacts.png`;
- сохранение фирменного акцента без больших сплошных заливок.

PR служебный. Код уже находится в `main`, в diff только этот документ.

# Контрольный запуск CI после многостороннего аудита

Дата: 2026-07-11.

Цель pull request — получить наблюдаемый запуск полного барьера проекта после реализации:

- уникальных итоговых `templateId`;
- раннего сбора runtime-ошибок;
- keyboard/focus выбора компоновки;
- защиты перехода `Частное → Фото СПН`;
- полного backup рабочего пространства;
- мобильного режима выполнения расклейки.

Обязательные проверки:

```bash
npm run validate
npm run smoke:browser
```

Документ не меняет runtime приложения и может остаться в истории проекта как запись о контрольной регрессии.

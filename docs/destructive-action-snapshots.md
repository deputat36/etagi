# Снимки перед разрушительными действиями

Генератор сохраняет одну последнюю резервную точку непосредственно перед действием, которое может заменить или удалить текущую работу.

## Когда создаётся снимок

Снимок создаётся перед:

- очисткой данных объекта;
- загрузкой выбранного именованного макета;
- загрузкой последнего макета из браузера;
- успешным импортом JSON-макета.

Ошибочный JSON не перезаписывает существующую резервную точку. Для загрузки именованного или последнего макета снимок не создаётся, если фактически нечего загружать.

## Что входит в снимок

Ключ хранения:

```text
etagi-raskleyka-destructive-snapshot-v1
```

В снимок входят:

- версия формата, уникальный идентификатор, дата и причина;
- все поля формы и переключатели состава макета;
- профиль СПН;
- служебные подписи контактов, бренда и отрывных полос;
- активная задача, шаблон, режим компоновки и количество копий;
- порядок блоков;
- QR;
- отображаемые фотографии в виде data URL;
- выбранный именованный макет и рабочий сценарий.

Сначала браузер пытается сохранить полный снимок. Если лимит localStorage не позволяет сохранить тяжёлые фотографии, сохраняется облегчённый вариант без фото с `photosOmitted: true`. Остальные данные не теряются.

## Границы текущего этапа

Этот этап создаёт надёжную резервную точку и отмечает её доступность через `data-destructive-snapshot-*` на `body`. Пользовательская команда восстановления будет добавлена следующим отдельным PR, чтобы не смешивать создание снимка и применение отката.

Хранится только одна последняя резервная точка. Новый успешный разрушительный сценарий заменяет предыдущую.

## Проверка

Статический контракт входит в общую проверку безопасности браузерного хранения:

```bash
npm run validate:storage-safety
npm run smoke:ui-actions
```

Сценарий `Destructive snapshot smoke` проверяет снимки перед очисткой, именованной загрузкой, загрузкой последнего макета и успешным импортом. Также он проверяет профиль, QR, служебную подпись, порядок блоков, количество копий, фото и защиту снимка от ошибочного импорта.

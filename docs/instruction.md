# Инструкция для СПН

1. Откройте генератор.
2. Выберите задачу: куплю, продам, новостройки, услуги, аренда, личный бренд или частное.
3. Выберите подходящий шаблон.
4. Замените телефон, имя, район, цену и параметры.
5. При необходимости загрузите фото.
6. Нажмите «Проверить».
7. Исправьте замечания.
8. Нажмите «Печать / PDF».

Для обычной расклейки чаще всего используйте 2 на А4. Для экономной массовой расклейки — 4 на А4. Для объекта с большим фото — 1 на А4.

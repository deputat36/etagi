# Проверка менеджера в интерфейсе

Документ описывает блок `assets/js/spnManagerReview.js`.

## Зачем нужен блок

Блок помогает быстро принять решение перед печатью макета, особенно если макет делает новичок или используется новый текст.

## Что проверяется

Менеджер отмечает:

- контакты заполнены;
- заголовок понятный;
- район, дом или ЖК указан;
- текст короткий и читаемый;
- формат А4 выбран правильно;
- контроль качества пройден.

## Как пользоваться

1. СПН готовит макет.
2. Нажимает «Проверить».
3. Исправляет критичные замечания.
4. Менеджер отмечает пункты в блоке проверки.
5. При необходимости пишет комментарий.
6. Нажимает «Скопировать решение».

## Когда проверка обязательна

Проверка менеджера обязательна:

- для первых макетов новичка;
- для новых текстов;
- для макетов с ипотекой;
- для новостроек;
- для формулировок про цены, сроки, бронь и юридические условия;
- для массовой печати.

## Решение

Если отмечены все пункты и нет критичных замечаний контроля качества, макет можно печатать.

Если часть пунктов не закрыта, макет нужно доработать до печати.

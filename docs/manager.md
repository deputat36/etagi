# Чек-лист менеджера

Документ для проверки макета СПН перед печатью.

## Проверить

1. Контакты заполнены.
2. Заголовок понятный.
3. Район или объект указан.
4. Текст короткий.
5. Формат А4 выбран правильно.
6. Макет прошёл контроль качества.

## Решение

Если всё понятно и читаемо, макет можно печатать. Если есть замечания контроля качества, сначала исправить их в генераторе.

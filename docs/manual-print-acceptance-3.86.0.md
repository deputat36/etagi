Статус: НЕ ПРОЙДЕНА

## Доступ к печати в длинной форме

Проверить сначала на рабочем ПК, затем на телефоне в вертикальной ориентации.

- [ ] На рабочем ПК форма не имеет горизонтальной прокрутки страницы.
- [ ] На телефоне шириной 360–430 px форма не имеет горизонтальной прокрутки страницы.
- [ ] Восемь шагов мастера читаются и нажимаются без увеличения масштаба браузера.
- [ ] Закреплённая панель предпросмотра не перекрывает важные поля и кнопки.
- [ ] Кнопка `К печати` видна и нажимается без увеличения масштаба.
- [ ] Одно нажатие `К печати` переводит к карточке печати без ручного поиска по длинной форме.
- [ ] После перехода кнопка `Проверить` находится в видимой области и получает заметный фокус.
- [ ] После проверки доступны `Печать / PDF`, настройки формата и цветности.
- [ ] Переход одинаково работает в режимах `Новичок`, `Быстро` и `Расширенно`.

## Обязательные форматы A4

### 1 на A4

- [ ] 1 на А4 без фото: текст не обрезан, поля ровные, телефон читается.
- [ ] В каждой карточке полный телефон и имя СПН.
- [ ] Восемь отрывных полос находятся внутри карточки и содержат полный номер.

### 2 на A4

- [ ] 2 на А4 с крупным телефоном: обе карточки одинаковые, линия деления корректна.
- [ ] Контакты и восемь отрывных полос каждой карточки не пересекают линию реза.
- [ ] Телефон читается без увеличения PDF.

### 3 на A4

- [ ] 3 на А4: три горизонтальные карточки одинаковой высоты и не накладываются.
- [ ] Контактный блок каждой карточки остаётся внутри границ.
- [ ] В каждой карточке восемь отрывных полос, полный телефон и читаемый заголовок.

### 4 на A4

- [ ] 4 на А4 в экономном цвете: контакты не накладываются, заливки светлые.
- [ ] Восемь отрывных полос каждой карточки остаются внутри границ.
- [ ] Линии реза проходят между карточками и не пересекают телефон.

### 6 на A4

- [ ] 6 на А4: шесть карточек 2×3 одинакового размера без обрезки.
- [ ] Отрывные полосы автоматически выключены и отсутствуют на листе.
- [ ] Полный телефон, имя СПН и контактный блок читаются в каждой карточке.

### 8 на A4

- [ ] 8 на А4: восемь карточек 2×4 находятся внутри листа.
- [ ] Отрывные полосы выключены и отсутствуют.
- [ ] Телефон, имя СПН и заголовок различимы без увеличения PDF.
- [ ] Компактный профиль не скрывает контактный блок и не создаёт overflow.

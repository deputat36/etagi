# Текущее состояние офисных функций

Документ фиксирует, что уже добавлено в генератор расклеек для офисного внедрения.

## Основные сценарии

- Быстрый выбор рабочей ситуации СПН.
- Офисные подборки шаблонов.
- Офисная разметка шаблонов ЖК «Теллерманов сад».
- Задание на расклейку для исполнителя.
- Отчёт после расклейки.
- История отчётов.
- Сводка истории отчётов.
- Проверка менеджера перед печатью.

## UI-helper модули

```text
assets/js/spnWizard.js                    быстрый подбор ситуации
assets/js/spnOfficeTemplateFilters.js     офисные подборки шаблонов
assets/js/spnDistributionTaskHelper.js    задание на расклейку
assets/js/spnDistributionReportHelper.js  отчёт, история, CSV и вывод по результату
assets/js/spnReportSummary.js             краткая сводка истории отчётов
assets/js/spnManagerReview.js             проверка менеджера перед печатью
```

## Шаблоны с офисной разметкой

```text
data/templates_tellerman_sad.json         ЖК «Теллерманов сад»
```

В пакете размечены подборки:

- рекомендовано;
- новичку;
- менеджер;
- Теллерманов сад.

## Что важно проверить вручную

1. Открывается ли генератор.
2. Видны ли офисные фильтры шаблонов.
3. Фильтр «Теллерманов сад» показывает локальные шаблоны.
4. Фильтр «Рекомендовано» показывает часть шаблонов ЖК.
5. Фильтр «Новичку» показывает безопасные первые макеты ЖК.
6. Фильтр «Менеджер» показывает ипотечные и чувствительные макеты.
7. Создаётся ли задание на расклейку.
8. Заполняется ли отчёт после расклейки.
9. Сохраняется ли отчёт в историю.
10. Появляется ли сводка истории.
11. Работает ли CSV-экспорт.
12. Виден ли блок проверки менеджера.
13. Сохраняются ли отметки проверки менеджера.
14. Копируется ли решение менеджера.

# Текущее состояние офисных функций

Документ фиксирует, что уже добавлено в генератор расклеек для офисного внедрения.

## Основные сценарии

- Быстрый выбор рабочей ситуации СПН.
- Офисные подборки шаблонов.
- Задание на расклейку для исполнителя.
- Отчёт после расклейки.
- История отчётов.
- Сводка истории отчётов.
- Проверка менеджера перед печатью.

## UI-helper модули

```text
assets/js/spnWizard.js                    быстрый подбор ситуации
assets/js/spnOfficeTemplateFilters.js     офисные подборки шаблонов
assets/js/spnDistributionTaskHelper.js    задание на расклейку
assets/js/spnDistributionReportHelper.js  отчёт, история, CSV и вывод по результату
assets/js/spnReportSummary.js             краткая сводка истории отчётов
assets/js/spnManagerReview.js             проверка менеджера перед печатью
```

## Что важно проверить вручную

1. Открывается ли генератор.
2. Видны ли офисные фильтры шаблонов.
3. Создаётся ли задание на расклейку.
4. Заполняется ли отчёт после расклейки.
5. Сохраняется ли отчёт в историю.
6. Появляется ли сводка истории.
7. Работает ли CSV-экспорт.
8. Виден ли блок проверки менеджера.
9. Сохраняются ли отметки проверки менеджера.
10. Копируется ли решение менеджера.

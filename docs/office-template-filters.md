# Офисные фильтры шаблонов

Документ описывает быстрые подборки шаблонов для СПН.

## Зачем нужны фильтры

Фильтры помогают специалисту начинать не с ручного поиска, а с рабочей ситуации.

## Подборки

### Рекомендовано

Для шаблонов, которые менеджер считает безопасными для регулярного использования.

### Новичку

Для первых макетов нового СПН. Такие шаблоны должны быть короткими, понятными и без сложных обещаний.

### Менеджер

Для макетов, которые лучше проверить перед массовой печатью.

### Теллерманов сад

Для локальных макетов по новостройке и раннему сбору интереса.

### Подъезд

Для коротких объявлений с отрывными телефонами.

## Как поддерживать

Чтобы шаблон попадал в подборку, добавьте нужные слова в `tags` шаблона:

```text
офис
рекомендовано
новичку
менеджер
подъезд
Теллерманов сад
```

Офисные фильтры используют обычный поиск, поэтому не требуют отдельной серверной логики.

## Что проверяется автоматически

Проверка офисных тегов встроена в общий валидатор шаблонов:

```bash
npm run validate:templates
```

Она контролирует разметку в трёх ключевых пакетах:

```text
data/templates_entrance.json
data/templates_borisoglebsk.json
data/templates_tellerman_sad.json
```

## Разметка ЖК «Теллерманов сад»

Пакет `data/templates_tellerman_sad.json` размечен офисными тегами.

- `bgo_newbuild_tellerman_waitlist` — офис, рекомендовано, новичку.
- `bgo_newbuild_tellerman_family` — офис, рекомендовано, новичку.
- `bgo_newbuild_tellerman_studio` — офис, рекомендовано, новичку.
- `bgo_newbuild_tellerman_mortgage` — офис, менеджер.
- `bgo_newbuild_tellerman_tradein` — офис, рекомендовано, менеджер.

Все шаблоны ЖК должны содержать теги `офис`, `Борисоглебск`, `Теллерманов сад` и один из тегов: `новичку` или `менеджер`.

Ипотечные, trade-in и другие чувствительные макеты лучше проверять менеджером до массовой печати.

## Подъездные шаблоны

Для `data/templates_entrance.json` действует правило:

```text
офис
рекомендовано
подъезд
новичку или менеджер
```

Простые шаблоны для поиска собственников, покупателей, аренды и оценки можно отдавать новичку.

Объектный подъездный шаблон `entrance_object_same_house` помечается как `менеджер`, потому что в нём нужно проверить объект, цену, фото, QR и формулировки перед массовой печатью.

## Локальные шаблоны Борисоглебска

Для `data/templates_borisoglebsk.json` действует правило:

```text
офис
рекомендовано
Борисоглебск
```

Безопасные шаблоны для поиска собственников, покупателей, новостроек и оценки дополнительно получают тег `новичку`.

Объектные шаблоны с фото и QR получают тег `менеджер`, а тег `новичку` для них запрещён.

Частные шаблоны без бренда не попадают в офисную рекомендованную подборку для новичка. Они должны иметь тег `менеджер`, чтобы перед печатью была проверка смысла и соответствия офисной политике.

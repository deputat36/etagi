# План офисной разметки A/B-шаблонов

Файл для будущего обновления `data/templates_ab_tests.json`.

## Логика

A/B-шаблоны — это тестовые гипотезы. Их не нужно все отдавать новичкам без контроля.

Безопасные консультационные тесты можно помечать как `рекомендовано` и `новичку`.

Прямые, частные и объектные тесты лучше помечать как `менеджер`.

## Карта тегов

```text
ab_owner_soft_price_check      офис, рекомендовано, новичку
ab_owner_direct_buyer          офис, менеджер
ab_owner_private_note          офис, менеджер
ab_object_neighbor_sold        офис, менеджер
ab_buyer_family_area           офис, рекомендовано
ab_service_after_viewing       офис, рекомендовано, новичку
```

## Статус

Прямое обновление `data/templates_ab_tests.json` через connector было заблокировано. Карта зафиксирована здесь для следующей безопасной итерации.

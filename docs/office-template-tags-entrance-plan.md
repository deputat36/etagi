## Ожидаемый результат

После разметки фильтры должны работать так:

- `Подъезд` — показывает все подъездные шаблоны.
- `Рекомендовано` — показывает безопасные подъездные шаблоны.
- `Новичку` — показывает простые первые подъездные макеты.
- `Менеджер` — показывает объектные и арендные подъездные макеты, которые лучше проверить.

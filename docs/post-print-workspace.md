# Компактный блок после печати

`Задание на расклейку` и `Отчёт после расклейки` объединены в один раскрываемый блок `После печати`.

Внутри сохраняются реальные рабочие разделы:

- задание на расклейку;
- режим исполнителя;
- отчёт и история;
- сводка менеджеру и аналитика.

## Зачем это сделано

До печати специалисту важнее текст, телефон, фото, QR, проверка и предпросмотр. Постоянно раскрытые задание и отчёт увеличивали длину формы и отвлекали от подготовки макета.

Новый блок по умолчанию закрыт, но все функции доступны одним нажатием.

## Безопасность

Модуль не создаёт копии и не переписывает содержимое. Он переносит существующие DOM-узлы `spnDistributionTask` и `spnDistributionReport` внутрь `<details>`, поэтому сохраняются обработчики, введённые данные, история отчётов и режим исполнителя.

Состояние раскрытия хранится локально:

```text
etagi-raskleyka-post-print-workspace-open-v1
```

Ключ входит в общий backup рабочего пространства.

## Wizard Flow

На шагах `Задание` и `Отчёт` блок раскрывается автоматически. Это временное открытие не перезаписывает выбор пользователя.

В режиме `Новичок` контейнер скрыт на остальных шагах, чтобы не перегружать мастер.

## Live-статус

В заголовке показывается:

- готово ли задание;
- заполнен ли текущий отчёт.

Статус обновляется при изменении места, количества листов, даты и данных отчёта.

## Ручная проверка

1. Открыть режим `Быстро` или `Расширенно`.
2. Найти блок `После печати` рядом с настройками печати.
3. Убедиться, что он по умолчанию закрыт.
4. Раскрыть блок и проверить наличие задания, режима исполнителя, отчёта и истории.
5. Заполнить место и количество листов — статус должен показать `задание готово`.
6. Заполнить дату, место и листы отчёта — статус должен показать `отчёт заполнен`.
7. Закрыть блок и перезагрузить страницу — состояние раскрытия должно восстановиться.
8. В режиме `Новичок` перейти на шаг `Задание`, затем `Отчёт` — блок должен открываться автоматически.
9. Вернуться на ранний шаг мастера — контейнер не должен занимать место.

## Автоматический контракт

```bash
npm run validate:post-print-workspace
```

Проверка запрещает клонирование разделов, глобальные click-перехваты, прямую печать и бесконечные таймеры.

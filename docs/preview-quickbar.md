# Закреплённая панель предпросмотра

Панель использует существующий блок `previewStatus` и не создаёт вторую дублирующую сводку.

## Что показывает

В верхней части предпросмотра остаются видны текущие показатели макета:

- количество макетов на А4;
- схема деления листа;
- наличие фото;
- цветность;
- сценарий и компоновка;
- число включённых блоков;
- настройки печати;
- качество после проверки.

Панель закрепляется внутри рабочей области при прокрутке. Статусы имеют `aria-live="polite"`, поэтому изменения доступны пользователям экранных дикторов без навязчивого прерывания.

## Кнопка «К печати»

Кнопка:

1. прокручивает левую панель к разделу печати;
2. переводит фокус на кнопку `Проверить`;
3. напоминает сначала проверить макет;
4. не вызывает `window.print()`;
5. не нажимает кнопку печати автоматически;
6. не обходит существующий print guard.

## Мобильный режим

На экранах до 640 пикселей панель перестраивается в одну колонку. Кнопка имеет минимальную высоту 44 пикселя, показатели остаются читаемыми, а управление масштабом переносится на отдельную строку.

## Ручная проверка

1. Открыть генератор в режиме `Быстро`.
2. Прокрутить предпросмотр вниз и убедиться, что верхняя панель остаётся видимой.
3. Изменить формат А4, фото и цветность — показатели должны обновиться.
4. Нажать `К печати`.
5. Убедиться, что левый блок прокрутился к разделу печати, а фокус получил `Проверить`.
6. Убедиться, что окно печати не открылось автоматически.
7. Повторить проверку на ширине 375–640 пикселей.

## Автоматический контракт

```bash
npm run validate:preview-quickbar
```

Проверка запрещает прямой вызов системной печати и глобальные обработчики кликов.

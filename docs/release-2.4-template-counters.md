# 2.4.0 — счётчики шаблонов

Добавлены счётчики в сценарные фильтры шаблонов.

Что изменилось:

- рядом с каждым сценарием теперь отображается количество доступных шаблонов;
- количество считается с учётом выбранной задачи, поиска, плотности и фильтра избранного;
- сценарии без доступных шаблонов становятся неактивными;
- под фильтрами появилась строка «Найдено шаблонов»;
- версия состояния обновлена до `2.4.0`.

Зачем это нужно:

СПН сразу видит, где есть подходящие шаблоны, и быстрее выбирает нужный сценарий без пустых переходов.

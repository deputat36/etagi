# 2.5.0 — проверка библиотеки шаблонов

Добавлен технический контроль качества шаблонов.

Что появилось:

- `tools/validate-templates.mjs` — скрипт проверки всех файлов `data/templates*.json`;
- `package.json` — команды `npm run validate` и `npm run validate:templates`;
- `.github/workflows/validate.yml` — автоматическая проверка через GitHub Actions;
- проверка обязательных полей шаблона;
- проверка обязательных полей внутри `data`;
- проверка допустимых значений `goal`, `density`, `photo`, `colorMode`, `splitMode`;
- предупреждения по длинным заголовкам, описаниям и дублям `id`.

Зачем это нужно:

Библиотека шаблонов быстро растёт. Теперь при добавлении новых JSON-файлов проще не сломать генератор: ошибки в структуре будут видны до использования шаблонов специалистами.

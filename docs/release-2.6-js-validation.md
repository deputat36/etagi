# 2.6.0 — проверка JavaScript

Добавлена проверка синтаксиса JavaScript-файлов.

Что появилось:

- `tools/validate-js.mjs` — скрипт проверки файлов `assets/js/*.js` и `tools/*.mjs`;
- команда `npm run validate:js`;
- общая команда `npm run validate` теперь запускает проверку шаблонов и проверку JavaScript;
- GitHub Actions workflow использует общую команду `npm run validate`, поэтому проверка JS будет запускаться вместе с проверкой шаблонов.

Зачем это нужно:

Проект стал модульным и в нём много JS-файлов. Проверка синтаксиса помогает поймать ошибку до того, как генератор перестанет открываться у специалистов.

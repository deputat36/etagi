# 2.7.0 — проверка связей файлов

Добавлена проверка статических связей проекта.

Что появилось:

- `tools/validate-assets.mjs` — скрипт проверки ссылок на файлы;
- команда `npm run validate:assets`;
- общая команда `npm run validate` теперь запускает проверку шаблонов, JavaScript и связей файлов;
- проверяются CSS и JS, подключённые в `index.html`;
- проверяются JSON-файлы шаблонов, подключённые в `assets/js/templates.js`;
- проверяются относительные `import` внутри модулей JavaScript.

Зачем это нужно:

При росте проекта легко забыть подключить файл, удалить нужный модуль или ошибиться в пути. Такая проверка заранее показывает, что генератор может не загрузить часть файлов на GitHub Pages.

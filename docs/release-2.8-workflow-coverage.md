# 2.8.0 — расширенная проверка GitHub Actions

Расширена зона автоматической проверки проекта.

Что изменилось:

- workflow переименован с `Validate templates` на `Validate project`;
- GitHub Actions теперь запускается не только при изменении шаблонов, но и при изменении `index.html` и `assets/**`;
- проверка покрывает шаблоны, JavaScript и связи файлов;
- версия `package.json` обновлена до `2.8.0`;
- версия состояния приложения обновлена до `2.8.0`.

Зачем это нужно:

Если изменить подключение CSS, JS, импорт модуля или файл шаблонов, автоматическая проверка сработает до того, как ошибка попадёт к специалистам на GitHub Pages.

# 2.9.0 — документация для развития проекта

Добавлена документация, чтобы проект было проще развивать без ошибок.

Что появилось:

- `docs/template-authoring-guide.md` — инструкция по добавлению новых шаблонов;
- `docs/maintenance-guide.md` — регламент сопровождения проекта;
- обновлён `README.md` с актуальной структурой, проверками и ссылками на документацию.

Зачем это нужно:

Генератор уже вырос из одного HTML-файла в модульный проект с несколькими пакетами шаблонов, проверками и workflow. Документация снижает риск случайно сломать шаблоны, подключения или публикацию на GitHub Pages.

Версию приложения не поднимали, потому что это документационное изменение без изменения пользовательского интерфейса.

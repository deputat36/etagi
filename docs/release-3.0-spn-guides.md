# 3.0.0 — инструкции для СПН и полевого теста

Добавлены пользовательские инструкции, чтобы генератор было проще внедрять в офисе.

Что появилось:

- `docs/spn-quick-start.md` — короткая инструкция для специалиста по недвижимости;
- `docs/field-test-checklist.md` — чек-лист проверки макетов после печати и расклейки;
- обновлён `README.md`, добавлены ссылки на новые документы.

Зачем это нужно:

Генератор должен быть не только технически рабочим, но и понятным для СПН. Быстрый старт помогает новому пользователю сделать первый макет без обучения, а чек-лист полевого теста помогает оценивать реальные отклики, а не только внешний вид объявления.

Версию приложения не поднимали, потому что это документационное изменение без изменения интерфейса.

# 3.1.0 — помощь внутри генератора

Добавлен блок помощи прямо в интерфейс генератора.

Что изменилось:

- в боковой панели появился раздел «Помощь»;
- добавлены быстрые ссылки на `docs/spn-quick-start.md` и `docs/field-test-checklist.md`;
- добавлены короткие подсказки по выбору формата для подъезда, объекта с фото и проверки перед печатью;
- добавлены стили для блока помощи;
- версия `package.json` обновлена до `3.1.0`;
- версия состояния приложения обновлена до `3.1.0`.

Зачем это нужно:

СПН не должен искать инструкцию отдельно. Теперь быстрый старт и чек-лист теста доступны прямо из генератора, рядом с сохранением и печатью.

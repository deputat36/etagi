# 3.10.0 — ссылки помощи в главном интерфейсе

Обновлён блок помощи в главном интерфейсе генератора.

Что изменилось:

- добавлена ссылка на `help/index.html` — центр помощи;
- добавлена ссылка на `help/trust.html` — доверие и возражения;
- в блоке помощи теперь доступны все пользовательские страницы;
- версия `package.json` обновлена до `3.9.0` ранее;
- версия состояния приложения обновлена до `3.9.0` ранее.

Зачем это нужно:

После появления новых страниц помощи важно, чтобы специалист мог открыть их не только по прямой ссылке, но и из самого генератора.

# 3.15.0 — план повторного контакта

Добавлена страница `help/follow-up.html`.

Она помогает СПН не потерять тёплый контакт после первого звонка по расклейке:

- план действий на 7 дней;
- что отправить собственнику;
- что отправить покупателю;
- готовые сообщения для повторного касания;
- что фиксировать после каждого контакта.

Страница добавлена в центр помощи `help/index.html`.

Версии `package.json` и `assets/js/state.js` обновлены до `3.15.0`.

# 3.18.0 — расширенные печатные пресеты

Добавлены новые варианты печати на А4:

- 1 крупно;
- 2 на А4;
- 3 полосы;
- 4 экономно;
- 6 мелко;
- 8 мини.

Зачем это нужно:

Рендер уже поддерживал разные сетки, но в интерфейсе были видны только базовые варианты. Теперь СПН может быстрее выбирать нестандартную плотность печати без ручной настройки.

README обновлён: добавлено описание расширенных печатных пресетов.

Версии `package.json` и `assets/js/state.js` обновлены до `3.18.0`.

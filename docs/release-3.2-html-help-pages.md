# 3.2.0 — HTML-страницы помощи

Добавлены отдельные HTML-страницы помощи для пользователей генератора.

Что появилось:

- `help/quick-start.html` — быстрый старт для СПН в удобном веб-формате;
- `help/field-test.html` — чек-лист полевого теста расклеек в удобном веб-формате;
- ссылки в блоке «Помощь» теперь ведут на HTML-страницы, а не на Markdown-файлы;
- версия `package.json` обновлена до `3.2.0`;
- версия состояния приложения обновлена до `3.2.0`.

Зачем это нужно:

Markdown-файлы полезны для разработки, но специалистам удобнее открывать готовые страницы с нормальным оформлением. Теперь помощь выглядит как часть продукта и корректно открывается на GitHub Pages.

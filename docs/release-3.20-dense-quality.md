# 3.20.0 — усиленная проверка плотных макетов

Контроль качества стал строже для плотной печати 6–8 макетов на А4.

Добавлены проверки:

- телефон слишком мелкий для плотной печати;
- заголовок слишком длинный для мини-макета;
- описание слишком длинное для 6–8 на А4;
- дополнительный блок перегружает мини-макет;
- слишком много преимуществ для мини-макета;
- плотная печать без безопасных полей;
- QR слишком мал для мини-макета.

Зачем это нужно:

После добавления пресетов 6 и 8 на А4 важно не дать СПН случайно напечатать красивый, но нечитаемый макет. Теперь генератор заранее подсказывает, что нужно укоротить, увеличить или убрать.

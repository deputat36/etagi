# 3.21.0 — стартовые шаблоны для нестандартных макетов

Добавлен файл `data/templates_custom.json`.

В него включены стартовые пустые шаблоны для ситуаций, когда СПН нужно не просто выбрать готовый текст, а быстро собрать нестандартный макет:

- пустой читаемый макет;
- пустой подъездный формат;
- объект с фото;
- частная записка;
- консультация;
- короткий A/B-тест.

Файл подключён в `assets/js/templates.js`.

Зачем это нужно:

Пользователь может найти шаблоны по словам `пустой`, `нестандартный`, `с нуля`, выбрать подходящую основу и быстро заменить текст под свою задачу без полной ручной сборки.

# 3.22.0 — инструкция по нестандартным макетам

Добавлена страница `help/custom-starters.html`.

Она объясняет, как использовать пустые стартовые шаблоны:

- когда выбирать пустой шаблон;
- как найти основу через поиск;
- какую основу выбрать под задачу;
- как быстро заменить текст;
- как не перегрузить нестандартный макет;
- как собрать заголовок, описание, преимущества и дополнительный блок.

Страница добавлена в центр помощи `help/index.html`.

Версии `package.json` и `assets/js/state.js` обновлены до `3.22.0`.

# 3.23.0 — адаптация без фото и улучшенные отрывные телефоны

Приняты замечания по макетам без фото и отрывным телефонам.

Изменения:

- если фото не загружено, блок фото больше не создаёт пустую серую область в середине макета;
- макет без фото получает класс `no-photo`, чтобы текстовые блоки могли лучше распределяться по высоте листа;
- для некомпактных макетов без фото увеличены описание, преимущества и дополнительный блок;
- отрывных телефонов стало 8 вместо 5;
- в каждом отрывном блоке добавлено слово `Недвижимость`, чтобы человек понимал, откуда номер;
- шрифт отрывных телефонов увеличен и усилен;
- для компактных макетов оставлены отдельные размеры, чтобы не ломать плотную печать.

Версии `package.json` и `assets/js/state.js` обновлены до `3.23.0`.

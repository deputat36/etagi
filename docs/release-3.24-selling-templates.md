# 3.24.0 — продающие шаблоны расклейки

Добавлен файл `data/templates_sales.json`.

В него включены продающие шаблоны для практической работы СПН:

- собственник думает о цене;
- есть покупатель;
- покупателю семейный вариант;
- объект с сильным фото;
- безопасная сделка;
- подъезд без давления.

Зачем это нужно:

Генератор должен не просто красиво раскладывать блоки, а помогать СПН делать расклейку, которая вызывает понятное действие: позвонить, уточнить цену, обсудить продажу, посмотреть объект или получить консультацию.

Шаблоны построены по практической логике:

- конкретный заголовок;
- понятная выгода;
- короткое описание;
- 2–3 причины позвонить;
- дополнительный смысловой блок;
- крупный телефон и отрывные контакты там, где это нужно.

Файл подключён в `assets/js/templates.js`.

Версии `package.json` и `assets/js/state.js` обновлены до `3.24.0`.

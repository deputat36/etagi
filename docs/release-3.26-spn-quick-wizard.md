# 3.26.0 — быстрый подбор для СПН

Добавлен быстрый подбор ситуации в интерфейсе выбора шаблонов.

Новые файлы:

- `assets/js/spnWizard.js`;
- `assets/css/spn-wizard.css`.

В интерфейсе появился блок `Быстрый подбор для СПН`.

Сценарии подбора:

- узнать цену у собственника;
- есть покупатель;
- подъезд / соседи;
- продать объект;
- найти покупателя;
- консультация / безопасность;
- нестандартный макет.

Как это работает:

СПН выбирает ситуацию, а инструмент сам переключает задачу и подставляет нужный поисковый запрос по шаблонам. Это уменьшает количество ручных действий и помогает быстрее выйти на подходящий продающий макет.

Зачем это нужно:

Инструмент должен быть не просто библиотекой шаблонов, а практическим помощником. Быстрый подбор помогает СПН начинать не с пустого поиска, а с понятной рабочей ситуации.

Версии `package.json` и `assets/js/state.js` обновлены до `3.26.0`.

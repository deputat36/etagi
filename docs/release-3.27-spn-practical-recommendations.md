# 3.27.0 — практические рекомендации в быстром подборе

Быстрый подбор для СПН теперь не только фильтрует шаблоны, но и даёт практическую рекомендацию по выбранной ситуации.

Добавлено:

- рекомендуемый формат печати;
- главный смысл макета;
- что отслеживать после расклейки;
- автоматическое применение рекомендуемого количества макетов на А4;
- автоматическое применение подходящего быстрого режима.

Примеры:

- `Есть покупатель` → 4 макета на А4, подъездный формат, отрывные контакты;
- `Продать объект` → 1 крупный макет, режим витрины, фото желательно;
- `Консультация / безопасность` → 2 макета на А4, спокойный текст, без перегруза.

Зачем это нужно:

СПН должен выбирать не настройки, а рабочую ситуацию. Инструмент сам подсказывает формат, смысл и метрику, чтобы расклейка была не просто красивой, а практической.

Версии `package.json` и `assets/js/state.js` обновлены до `3.27.0`.

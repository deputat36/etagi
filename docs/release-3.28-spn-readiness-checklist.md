# 3.28.0 — практическая готовность макета

Добавлен отдельный блок `Практическая готовность`.

Новые файлы:

- `assets/js/spnChecklist.js`;
- `assets/css/spn-checklist.css`.

Чек-лист проверяет, готов ли макет к реальной расклейке с точки зрения СПН:

- телефон заполнен и виден;
- есть цепляющий заголовок;
- понятно, к кому обращаемся;
- описание короткое и по делу;
- есть понятный призыв;
- есть 2–3 причины откликнуться;
- снято опасение человека.

Зачем это нужно:

СПН видит не только технический контроль качества, но и практическую готовность макета. Это помогает быстрее довести расклейку до состояния, когда её не стыдно печатать и есть шанс получить отклик.

Версии `package.json` и `assets/js/state.js` обновлены до `3.28.0`.

# 3.30.0 — план расклейки

Добавлен блок `План расклейки`.

Новые файлы:

- `assets/js/spnFieldPlan.js`;
- `assets/css/spn-field-plan.css`.

Блок автоматически формирует практический план для СПН:

- формат печати;
- объём первого теста;
- где клеить;
- что считать;
- когда смотреть результат;
- какой следующий шаг делать после теста.

Также добавлены быстрые ссылки:

- на лист учёта откликов;
- на инструкцию по анализу результатов.

Есть кнопка `Скопировать`, чтобы СПН мог забрать план в заметки, чат или отчёт.

Зачем это нужно:

Генератор должен вести СПН дальше печати. Теперь инструмент помогает не только сделать макет, но и провести нормальный полевой тест: распечатать, расклеить, замерить отклики и принять решение, что менять.

Версии `package.json` и `assets/js/state.js` обновлены до `3.30.0`.

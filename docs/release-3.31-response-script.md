# 3.31.0 — скрипт обработки отклика

В блок `План расклейки` добавлен раздел `Что говорить по отклику`.

Теперь инструмент автоматически формирует короткий скрипт разговора под текущий макет и ситуацию.

Скрипт включает:

- начало разговора;
- уточняющий вопрос;
- какую пользу дать человеку;
- следующий шаг.

Логика меняется в зависимости от смысла макета:

- оценка цены;
- есть покупатель;
- объект с фото / продажа объекта;
- безопасная сделка / консультация;
- универсальный сценарий.

Добавлена кнопка `Скопировать скрипт`, чтобы СПН мог быстро перенести текст в заметки, чат или обучающие материалы.

Зачем это нужно:

Расклейка должна не только приводить отклик, но и помогать правильно обработать этот отклик. Теперь генератор поддерживает СПН на следующем шаге после звонка или сообщения.

Версии `package.json` и `assets/js/state.js` обновлены до `3.31.0`.

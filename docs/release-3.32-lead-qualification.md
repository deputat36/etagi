# 3.32.0 — квалификация отклика

Добавлен блок `Квалификация отклика`.

Новые файлы:

- `assets/js/spnLeadQualification.js`;
- `assets/css/spn-lead-qualification.css`.

Блок помогает СПН быстро оценить качество обращения после расклейки:

- горячий отклик;
- тёплый отклик;
- холодный отклик.

Для каждого типа отклика показываются:

- признаки;
- вопросы, которые нужно задать;
- рекомендуемый следующий шаг.

Также добавлено поле короткой заметки и кнопка `Скопировать заметку`, чтобы СПН мог быстро перенести результат разговора в заметки, чат, CRM или рабочий отчёт.

Зачем это нужно:

Расклейка должна оцениваться не только по количеству звонков, но и по качеству откликов. Новый блок помогает фиксировать, какие макеты дают реальные рабочие контакты, а какие дают слабые или случайные обращения.

Версии `package.json` и `assets/js/state.js` обновлены до `3.32.0`.

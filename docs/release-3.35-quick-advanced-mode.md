# 3.35.0 — быстрый и расширенный режим интерфейса

По итогам аудита добавлен переключатель режима работы интерфейса.

Новые файлы:

- `assets/js/spnUiMode.js`;
- `assets/css/spn-ui-mode.css`.

## Быстрый режим

Быстрый режим включается по умолчанию и показывает главное для подготовки макета:

- выбор задачи;
- быстрый подбор ситуации;
- шаблоны;
- данные СПН и объекта;
- фото / QR;
- печать;
- практическая готовность;
- контроль качества.

В быстром режиме скрываются второстепенные блоки:

- гибкая настройка для опытных;
- сохранение;
- помощь;
- план расклейки;
- квалификация отклика;
- итог теста расклейки;
- управление составом и порядком блоков.

## Расширенный режим

Расширенный режим показывает всё:

- тонкие настройки;
- сохранение макетов;
- инструменты после расклейки;
- квалификацию откликов;
- оценку результата;
- помощь и дополнительные материалы.

## Зачем это нужно

Аудит показал, что инструмент стал функциональным, но перегруженным. Новый режим снижает порог входа для СПН: новичок видит только главное, а опытный пользователь может включить все возможности.

Версии `package.json` и `assets/js/state.js` обновлены до `3.35.0`.

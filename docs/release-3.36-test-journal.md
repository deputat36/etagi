# 3.36.0 — журнал тестов расклейки

Добавлен локальный журнал тестов в блок `Итог теста расклейки`.

Изменены файлы:

- `assets/js/spnResultEstimator.js`;
- `assets/css/spn-result-estimator.css`.

Что добавлено:

- кнопка `Сохранить тест`;
- сохранение последних 30 тестов в LocalStorage браузера;
- список последних 5 тестов в интерфейсе;
- кнопка `Скопировать журнал`;
- кнопка `Очистить журнал`.

В журнал сохраняется:

- дата;
- источник / текущий макет;
- заголовок;
- формат печати;
- количество листов;
- количество откликов;
- горячие и тёплые отклики;
- процент отклика;
- качество отклика;
- вывод;
- следующий шаг.

Зачем это нужно:

Теперь СПН и менеджер могут сравнивать тесты между собой, а не просто копировать отдельный итог. Это закрывает следующий этап плана аудита: накопление данных по расклейке и понимание, какие макеты реально работают.

Версии `package.json` и `assets/js/state.js` обновлены до `3.36.0`.

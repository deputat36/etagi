# 3.37.0 — финальная готовность к печати

В блок `Практическая готовность` добавлена финальная проверка перед печатью.

Изменены файлы:

- `assets/js/spnChecklist.js`;
- `assets/css/spn-checklist.css`.

Что добавлено:

- блок `Можно готовить к печати` / `Перед печатью есть критичные пункты`;
- разделение замечаний на критичные и предупреждения;
- кнопка `Проверить`;
- кнопка `Печать / PDF`, которая активна только если нет критичных пунктов.

Критичные пункты:

- нет телефона;
- нет нормального заголовка;
- нет контекста;
- нет описания.

Предупреждения:

- нет призыва;
- мало причин откликнуться;
- нет снятия опасений;
- для массовой расклейки выключены отрывные телефоны;
- QR включён, но ссылка не заполнена.

Зачем это нужно:

СПН должен понимать, можно ли уже печатать макет. Новая финальная проверка снижает риск, что специалист распечатает слабую расклейку без телефона, контекста или понятного смысла.

Версии `package.json` и `assets/js/state.js` обновлены до `3.37.0`.

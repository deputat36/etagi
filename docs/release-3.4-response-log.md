# 3.4.0 — лист учёта откликов

Добавлен инструмент для полевого учёта результатов расклейки.

Что появилось:

- `help/response-log.html` — печатный лист учёта откликов;
- в блоке «Помощь» добавлена ссылка «Лист учёта откликов»;
- на странице есть кнопка «Печать / PDF»;
- добавлены поля для данных теста, таблица обращений и блок итогов;
- версия `package.json` обновлена до `3.4.0`;
- версия состояния приложения обновлена до `3.4.0`.

Зачем это нужно:

Теперь СПН может не только создать и распечатать макет, но и зафиксировать, сколько обращений дала конкретная расклейка, какие лиды были целевыми и что нужно изменить в следующем тесте.

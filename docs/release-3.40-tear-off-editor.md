# 3.40.0 — редактирование отрывных листочков

Добавлено редактирование надписи на отрывном листочке.

Новые файлы:

- `assets/js/spnTearOffEditor.js`.

Изменены файлы:

- `assets/js/render.js`;
- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- поле `Надпись на отрывном листе`;
- быстрые варианты: `Недвижимость`, `Куплю квартиру`, `Оценка цены`, `Есть покупатель`, `Продажа дома`, `Консультация`;
- сохранение выбранной надписи в браузере;
- автоматическое обновление предпросмотра при изменении надписи;
- раздельный вывод надписи и номера телефона в отрывном листочке.

Зачем это нужно:

Раньше надпись была фиксированной: `Недвижимость`. Теперь СПН может адаптировать отрывной блок под задачу макета. Например, для поиска объекта можно поставить `Куплю квартиру`, для оценки — `Оценка цены`, для консультации — `Консультация`.

Номер телефона остаётся отдельным крупным элементом, чтобы не терять читаемость.

Версии `package.json` и `assets/js/state.js` обновлены до `3.40.0`.

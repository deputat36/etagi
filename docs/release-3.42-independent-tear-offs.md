# 3.42.0 — независимый блок отрывных листочков

Доработана логика отрывных листочков.

Изменены файлы:

- `assets/js/render.js`;
- `assets/js/spnChecklist.js`;
- `package.json`;
- `assets/js/state.js`.

Что изменилось:

- отрывные листочки больше не зависят от блока `Контакты`;
- если блок `Контакты` выключен, но `Отрывные` включены — отрывные листочки всё равно выводятся;
- телефон теперь считается видимым, если включены либо контакты, либо отрывные листочки;
- финальная проверка перед печатью больше не ругается на скрытые контакты, если телефон есть на отрывных листочках.

Зачем это нужно:

Пользователь может собрать компактную расклейку: убрать большой блок контактов, оставить только отрывные телефоны и получить больше места для заголовка, описания или фото. Раньше при выключении контактов отрывные листочки тоже пропадали. Теперь это отдельный независимый блок.

Версии `package.json` и `assets/js/state.js` обновлены до `3.42.0`.

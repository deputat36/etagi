# 3.45.0 — редактирование брендовой строки

Добавлено редактирование текстовой части брендовой строки.

Новый файл:

- `assets/js/spnBrandEditor.js`.

Изменены файлы:

- `assets/js/render.js`;
- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- поле `Текст рядом с логотипом`;
- поле `Подпись справа`;
- быстрые варианты для брендовой строки;
- сохранение выбранных значений в браузере;
- автоматическое обновление предпросмотра.

Быстрые варианты:

- `Этажи`;
- `Этажи Борисоглебск`;
- `Агентство Этажи`;
- `etagi.com`;
- `Борисоглебск`;
- `Недвижимость`.

Зачем это нужно:

Блок логотипа уже можно было включать и выключать. Теперь его можно адаптировать под задачу макета: оставить короткое `Этажи`, сделать локальный акцент `Этажи Борисоглебск` или заменить правую подпись на город / направление. Красная метка `Э` остаётся фирменным якорем макета.

Версии `package.json` и `assets/js/state.js` обновлены до `3.45.0`.

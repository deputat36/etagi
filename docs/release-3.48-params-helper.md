# 3.48.0 — помощник параметров объекта

Добавлен помощник для поля `Параметры`.

Новый файл:

- `assets/js/spnParamsHelper.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- быстрые параметры для квартир;
- быстрые параметры для домов;
- быстрые параметры для участков;
- быстрые параметры для коммерции;
- автоматический подбор вариантов по типу объекта;
- кнопка `Не показывать`, которая скрывает блок параметров на макете;
- автоматическое включение блока параметров при выборе варианта.

Примеры формулировок:

- `1-комн., 35 м², 3/5 эт.`;
- `2-комн., 48 м², 2/5 эт.`;
- `дом 80 м², участок 6 сот.`;
- `6 сот., ИЖС`;
- `1 этаж, отдельный вход`;
- `ремонт, мебель, техника`.

Зачем это нужно:

Параметры объекта часто пишутся по-разному и занимают лишнее место. Новый помощник даёт короткие готовые формулировки и подстраивается под тип объекта. Это ускоряет заполнение макета и делает блок параметров более читаемым.

Версии `package.json` и `assets/js/state.js` обновлены до `3.48.0`.

# 3.49.0 — помощник района и адреса

Добавлен помощник для поля `Район / адрес / ЖК`.

Новый файл:

- `assets/js/spnAreaHelper.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- быстрые варианты районов и улиц;
- дополнительные контекстные признаки района;
- кнопка `В заголовок`, которая добавляет район в заголовок;
- автоматическое включение блока параметров при выборе района;
- визуальное состояние, если район не заполнен или блок параметров скрыт.

Быстрые варианты:

- `Центр`;
- `Северный`;
- `Юго-Восточный`;
- `район Аэродромной`;
- `ул. Бланская`;
- `ул. Советская`;
- `ул. Пешкова`;
- `Борисоглебск`.

Дополнительный контекст:

- `рядом школа`;
- `тихий двор`;
- `центр рядом`;
- `остановка рядом`;
- `удобный заезд`;
- `обжитой район`.

Зачем это нужно:

Район и адрес — один из главных крючков локальной расклейки. Человек быстрее реагирует, если видит знакомую улицу, район или ситуацию рядом с домом. Новый помощник ускоряет заполнение и помогает вынести локацию в заголовок.

Версии `package.json` и `assets/js/state.js` обновлены до `3.49.0`.

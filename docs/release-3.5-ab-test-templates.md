# 3.5.0 — шаблоны для A/B-тестов расклейки

Добавлен отдельный пакет шаблонов для проверки разных гипотез по откликам.

Что появилось:

- `data/templates_ab_tests.json` — новый пакет A/B-тестовых шаблонов;
- `assets/js/templates.js` теперь подключает новый пакет;
- добавлено 6 шаблонов для тестов: мягкая оценка цены, прямой покупатель, частная записка, продажа рядом, семейный покупатель, консультация перед продажей;
- версия `package.json` обновлена до `3.5.0`;
- версия состояния приложения обновлена до `3.5.0`.

Зачем это нужно:

Теперь можно не просто делать красивые объявления, а тестировать разные заходы: мягкий, прямой, частный, объектный и консультационный. Это поможет понять, какие формулировки реально дают обращения в конкретном районе.

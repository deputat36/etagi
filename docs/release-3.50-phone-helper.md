# 3.50.0 — помощник телефона для отклика

Добавлен помощник для поля `Телефон`.

Новый файл:

- `assets/js/spnPhoneHelper.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- форматирование российского номера телефона;
- визуальный статус телефона: номер не заполнен, номер скрыт, виден везде, только контакты, только отрывные;
- кнопка `Форматировать`;
- кнопка `Везде показать`;
- кнопка `Только отрывные`.

Сценарии:

- если номер введён как `89003006454`, он приводится к формату `+7 900 300-64-54`;
- если нужно максимум откликов, можно одним нажатием включить телефон и в контактах, и на отрывных листочках;
- если нужен компактный макет, можно оставить телефон только на отрывных листочках.

Зачем это нужно:

Телефон — главный элемент отклика. Если он не заполнен или скрыт, расклейка не работает. Новый помощник помогает быстро проверить номер и выбрать, где он будет виден на макете.

Версии `package.json` и `assets/js/state.js` обновлены до `3.50.0`.

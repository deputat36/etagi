# 3.51.0 — помощник подписи специалиста

Добавлен помощник для поля `Имя СПН`.

Новый файл:

- `assets/js/spnAgentHelper.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- быстрые варианты подписи специалиста;
- кнопка `Привести имя`, которая аккуратно форматирует введённое имя;
- автоматическая генерация вариантов на основе имени;
- обновление предпросмотра через существующее поле `Имя СПН`.

Примеры вариантов:

- `Алексей`;
- `Алексей, специалист по недвижимости`;
- `Алексей, Этажи Борисоглебск`;
- `Специалист по недвижимости`;
- `Ваш специалист по недвижимости`;
- `Команда Этажи Борисоглебск`.

Зачем это нужно:

Строка под телефоном влияет на доверие. В одних макетах достаточно имени, в других лучше добавить роль или локальную привязку. Новый помощник позволяет быстро выбрать подходящую подпись без ручного набора.

Версии `package.json` и `assets/js/state.js` обновлены до `3.51.0`.

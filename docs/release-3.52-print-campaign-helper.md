# 3.52.0 — сценарии печати

Добавлен помощник сценариев печати.

Новый файл:

- `assets/js/spnPrintCampaignHelper.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- быстрый сценарий `Тест 2 листа`;
- быстрый сценарий `Подъезд`;
- быстрый сценарий `Массовая расклейка`;
- быстрый сценарий `Объект с фото`;
- быстрый сценарий `Стенд / офис`;
- быстрый сценарий `Проверка печати`.

Что меняет сценарий:

- количество макетов на А4;
- деление листа;
- отрывные телефоны;
- линии реза;
- безопасные поля;
- режим проверки печати;
- фото;
- QR.

Зачем это нужно:

СПН часто думает не категориями настроек, а задачами: сделать тестовую расклейку, подготовить подъезд, напечатать массовый вариант или вывести крупный макет на стенд. Новый помощник позволяет выбрать сценарий и сразу получить подходящие настройки печати.

Версии `package.json` и `assets/js/state.js` обновлены до `3.52.0`.

# 3.53.0 — задание на расклейку

Добавлен помощник `Задание на расклейку`.

Новый файл:

- `assets/js/spnDistributionTaskHelper.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- поле количества листов;
- поле места расклейки;
- поле ответственного;
- поле срока выполнения;
- автоматический расчёт примерного количества объявлений;
- текст задания, который можно скопировать одной кнопкой;
- сохранение параметров задания в браузере.

Что попадает в задание:

- заголовок макета;
- локация;
- тип объекта / задача;
- количество листов;
- количество объявлений на А4;
- примерный общий объём объявлений;
- настройки печати;
- телефон для отклика;
- ответственный;
- срок;
- напоминание зафиксировать адреса, количество листов и первые отклики.

Зачем это нужно:

После подготовки макета часто нужно передать задачу помощнику или самому СПН: где клеить, сколько листов, какой срок, что потом зафиксировать. Новый блок превращает макет не только в печатный файл, но и в понятное рабочее задание.

Версии `package.json` и `assets/js/state.js` обновлены до `3.53.0`.

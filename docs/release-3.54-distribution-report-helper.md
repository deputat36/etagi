# 3.54.0 — отчёт после расклейки

Добавлен помощник `Отчёт после расклейки`.

Новый файл:

- `assets/js/spnDistributionReportHelper.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- дата расклейки;
- место расклейки;
- количество расклеенных листов;
- количество звонков и сообщений;
- количество целевых обращений;
- заметки по локации;
- следующий шаг;
- кнопка заполнения из задания;
- кнопка копирования отчёта;
- примерный расчёт отклика.

Что попадает в отчёт:

- дата;
- макет;
- локация;
- количество листов;
- примерное количество объявлений;
- звонки и сообщения;
- целевые обращения;
- примерный процент отклика;
- заметки;
- следующий шаг.

Зачем это нужно:

Расклейка должна давать не только листовки, но и управляемый результат. После каждой расклейки СПН может быстро зафиксировать, где клеили, сколько было откликов и что делать дальше: повторить, изменить текст, сменить район или усилить оффер.

Версии `package.json` и `assets/js/state.js` обновлены до `3.54.0`.

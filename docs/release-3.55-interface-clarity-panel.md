# 3.55.0 — понятный маршрут интерфейса

Добавлена панель `Маршрут работы`.

Новый файл:

- `assets/js/spnClarityPanel.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- верхняя липкая панель с понятным маршрутом работы;
- шаги: `Задача`, `Шаблон`, `Данные`, `Фото/QR`, `Печать`, `Проверка`;
- счётчик готовности шагов;
- подсказка следующего действия;
- кнопка `Перейти`, которая прокручивает к нужному месту;
- кнопка `Скрыть подсказки` / `Показать подсказки`;
- сохранение состояния скрытых подсказок в браузере;
- визуальное выделение блока, к которому перешёл пользователь.

Зачем это нужно:

После добавления множества помощников интерфейс стал функциональным, но более плотным. Новая панель делает работу понятнее: показывает, что уже заполнено, что нужно сделать дальше, и позволяет временно скрыть вспомогательные блоки, чтобы оставить только основные поля и предпросмотр.

Какие вспомогательные блоки можно скрыть:

- помощник отрывных листочков;
- помощник бренда;
- помощник призыва в контактах;
- помощник QR;
- помощник цены;
- помощник параметров;
- помощник района;
- помощник телефона;
- помощник подписи специалиста;
- сценарии печати;
- задание на расклейку;
- отчёт после расклейки.

Версии `package.json` и `assets/js/state.js` обновлены до `3.55.0`.

# 3.56.0 — компактное меню шаблонов

Исправлена проблема с большими карточками шаблонов в меню выбора.

Новый файл:

- `assets/js/spnTemplateMenuCompact.js`.

Изменены файлы:

- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что добавлено:

- компактный список шаблонов по умолчанию;
- скрытие декоративной мини-карточки макета в меню;
- более плотное отображение заголовка, описания и тегов;
- больше шаблонов видно без прокрутки;
- переключатель `Компактно` / `С мини-макетом`;
- сохранение выбранного режима меню в браузере.

Зачем это нужно:

Мини-карточка шаблона занимала много места, но почти не помогала выбрать подходящий вариант. Теперь список работает как рабочее меню: пользователь быстрее видит названия, смысл шаблонов и теги, а при необходимости может вернуть старый визуальный режим.

Версии `package.json` и `assets/js/state.js` обновлены до `3.56.0`.

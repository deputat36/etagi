# 3.59.0 — компактный блок параметров

Добавлен компактный режим блока параметров для плотной печати.

Новые файлы:

- `assets/css/meta-compact.css`;
- `assets/js/spnMetaCompactStyle.js`.

Изменены файлы:

- `assets/js/render.js`;
- `assets/js/spnUiMode.js`;
- `package.json`;
- `assets/js/state.js`.

Что изменено:

- для 4, 6 и 8 макетов на А4 район, тип и параметры теперь выводятся компактной строкой;
- для обычных и крупных макетов остаются прежние карточки параметров;
- компактный вариант включается автоматически при `printCount >= 4` или `layoutDensity = dense`;
- новый CSS подключается через отдельный модуль, чтобы не раздувать основной `app.css`.

Зачем это нужно:

Раньше блок параметров занимал слишком много высоты в плотных макетах. Для подъездной и массовой расклейки важнее сохранить читаемость заголовка и телефона. Теперь локальная привязка не исчезает, но занимает меньше места.

Версии `package.json` и `assets/js/state.js` обновлены до `3.59.0`.

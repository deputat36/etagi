# 3.8.0 — страница помощи по доверию

Добавлена отдельная HTML-страница помощи `help/trust.html`.

Страница объясняет:

- когда использовать мягкие доверительные шаблоны;
- какие возражения закрывать в тексте;
- какие формулировки работают лучше;
- чего лучше избегать;
- как сравнивать прямой и доверительный подходы.

Обновлены версии `package.json` и `assets/js/state.js` до `3.8.0`.

- В `index.html` обновлён cache-busting параметр подключения `spnUiMode.js` до `newbie-wizard-20260703-1`.
- Добавлен ручной чек-лист `docs/newbie-mode-regression-checklist.md` для проверки режима `Новичок` после изменений.
- Добавлена автоматическая проверка `validate:newbie-mode-docs`, которая контролирует ключевые newbie-модули, чек-лист и описание режима в README.
- Регламент сопровождения `docs/maintenance-guide.md` дополнен правилами безопасной работы с режимом `Новичок`.
- Версия состояния и пакета обновлена до `3.85.0`.

# Быстрый старт для СПН

Инструкция для специалиста по недвижимости, который хочет быстро сделать расклейку и не разбираться в технических настройках.

## За 3 минуты

1. Откройте генератор.
2. Выберите задачу: куплю, продам, новостройки, аренда, услуги или личный бренд.
3. Выберите шаблон.
4. Введите имя и телефон.
5. Заполните район, цену, параметры и текст.
6. Нажмите «Проверить».
7. Нажмите «Печать / PDF».

## Что заполнить обязательно

Минимум для рабочего объявления:

```text
Имя СПН
Телефон
Заголовок
Описание
Район или адрес
```

Если нет времени, лучше сделать коротко, но понятно. Телефон должен быть крупным.

## Как выбрать шаблон

### Для поиска собственника

Выберите задачу «Куплю / найти продавца».

Подходящие сценарии:

```text
Собственники
Подъезд
Экономно
```

Лучше использовать 2 или 4 объявления на А4.

### Для расклейки в подъезде

Выберите сценарий «Подъезд».

Лучше включить:

```text
Отрывные телефоны
Линии реза
Безопасные поля
```

Текст должен быть очень коротким. Человек читает объявление быстро, стоя у двери или лифта.

### Для продажи объекта

Выберите задачу «Продать объект».

Если есть хорошее фото, используйте сценарий «С фото».

Для витрины или стойки лучше 1 объявление на А4. Для расклейки — 2 на А4.

### Для частного стиля

Выберите задачу «Частное» или цветность «Частное без бренда».

Такой формат подходит, если нужно сделать объявление мягче и без явной фирменной подачи.

## Как писать заголовок

Хороший заголовок короткий и прямой:

```text
КУПЛЮ КВАРТИРУ
ПРОДАЁТСЯ ДОМ
СКОЛЬКО СТОИТ ВАША КВАРТИРА?
ХОЧУ КВАРТИРУ В ЭТОМ ДОМЕ
```

Плохой заголовок слишком длинный:

```text
Специалист по недвижимости поможет вам выгодно продать квартиру в вашем районе
```

## Как писать описание

Описание должно отвечать на вопрос: почему человеку стоит позвонить?

Пример:

```text
Есть покупатель на квартиру в этом районе. Состояние обсуждается. Подскажу реальную цену и варианты продажи.
```

Лучше не писать слишком много. Если текст не помещается, нажмите «Сделать короче».

## Когда включать QR

QR полезен, если:

- макет крупный;
- есть ссылка на объект;
- есть ссылка на подборку;
- нужно вести человека в мессенджер или на страницу СПН.

QR не обязателен для маленьких подъездных объявлений на 4 макета на А4. Там важнее крупный телефон.

## Как сохранить свой профиль

1. Введите имя и телефон.
2. Нажмите «Сохранить профиль СПН».
3. После этого можно менять шаблоны, а имя и телефон будут сохраняться.

## Как сохранить удачный макет

1. Заполните название макета.
2. Нажмите «Сохранить как макет».
3. Позже выберите его из списка и нажмите «Загрузить выбранный».

Фото в сохранённый макет не попадают, чтобы не переполнять память браузера.

## Перед печатью

Проверьте:

```text
Телефон написан правильно
Заголовок читается с расстояния
Нет лишнего текста
Включены линии реза
Макет помещается на лист
QR сканируется, если он есть
```

Если есть критичные ошибки, генератор покажет их в блоке «Контроль качества».

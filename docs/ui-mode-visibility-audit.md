# Матрица видимости режимов интерфейса

Дата проверки: 22.07.2026.

Матрица фиксирует ожидаемую видимость основных рабочих блоков. Она нужна, чтобы упрощение интерфейса не превращалось в случайное удаление функций.

| Блок | Быстро | Новичок | Расширенно |
|---|---|---|---|
| Рабочая ситуация | видна | видна | видна |
| Ручной выбор технической задачи | скрыт | скрыт | виден |
| Фильтр плотности шаблонов | виден | скрыт | виден |
| Быстрые режимы компоновки | видны | видны | видны |
| Ручной состав и порядок блоков | скрыт | скрыт | виден |
| Фото и QR | видны | видны по шагу | видны |
| Основные настройки печати | видны | видны по шагу | видны |
| Деление листа и режим проверки печати | видны | скрыты | видны |
| Контроль качества | виден | виден по шагу | виден |
| Сохранение и перенос | скрыты | скрыты | видны |
| Помощь | скрыта | компактно: быстрый старт и FAQ | полностью |
| Гибкая настройка для опытных | скрыта | скрыта | видна |
| После печати | компактно, закрыто | открывается на шагах задания и отчёта | компактно, закрыто |
| Дополнительная аналитика | скрыта | скрыта | компактно, закрыта |

## Решение текущей итерации

Полное скрытие помощи у новичка признано ошибочным: пользователь терял быстрый старт именно в режиме обучения. Полный центр помощи также не подходит, потому что семь ссылок и советы перегружают пошаговый путь. Поэтому новичку доступны только быстрый старт и FAQ.

## Следующий кандидат для отдельной проверки

`Сохранение и перенос` полностью скрыто в режиме `Быстро`. Перед изменением необходимо отдельно определить минимальное действие после подготовки макета: локальное сохранение, файл одного макета или переход в `Расширенно`. Нельзя возвращать весь большой блок без проверки пользовательского сценария.

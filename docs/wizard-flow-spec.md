# Wizard Flow v1

## Цель

Упростить создание макета расклейки. Пользователь должен идти по шагам, а не видеть все настройки сразу.

## Новый порядок работы

```text
1. Цель
2. Количество объявлений на А4
3. Категория заготовок
4. Выбор макета
5. Редактирование текста
6. Фото и QR
7. Проверка
8. Печать / PDF
9. Задание на расклейку
10. Отчёт по результату
```

## Шаг 1. Цель

Показывать крупные карточки:

- найти собственника;
- найти покупателя;
- продать объект;
- новостройка;
- оценка / услуга;
- аренда;
- личный бренд;
- частное объявление.

## Шаг 2. Количество объявлений на А4

Этот выбор должен быть в начале.

Варианты:

- 1 крупно;
- 2 на А4 — чаще всего;
- 4 экономно;
- 6 мелко;
- 8 мини.

По умолчанию должен быть выбран вариант `2 на А4`.

## Шаг 3. Категория заготовок

Категории:

- рекомендовано;
- новичку;
- подъезд;
- собственники;
- покупатели;
- объекты;
- новостройки;
- Теллерманов сад;
- доверие;
- с фото;
- пустые под проверку менеджера.

## Шаг 4. Выбор макета

Карточка макета должна показывать:

- мини-превью;
- название;
- назначение;
- формат листа;
- метки: новичку, менеджер, подъезд, с фото.

## Шаг 5. Редактирование

Каждый видимый текст в макете должен быть редактируемым.

Основные группы полей:

- контакты;
- объект / район;
- текст объявления;
- дополнительный блок.

Сложные настройки скрыть в блок `Дополнительно`.

## Шаг 6. Фото и QR

Показывать фото и QR только когда они нужны выбранному макету.

Для фото нужны отдельные компоновки:

- крупное фото сверху;
- фото слева;
- карточка объекта с фото;
- планировка;
- изображение ЖК.

Фото не должно выглядеть узкой полоской.

## Шаг 7. Проверка

Показывать:

- ошибки;
- важные замечания;
- советы;
- проверку менеджера, если шаблон требует контроля.

## Шаг 8. Печать

Показывать только настройки печати:

- количество объявлений на листе;
- деление листа;
- линии реза;
- безопасные поля;
- отрывные телефоны;
- печать / PDF.

## Режимы

### Новичок СПН

Показывать только безопасные шаблоны и основные поля.

### Опытный СПН

Оставить возможность открыть дополнительные настройки.

### Менеджер

Сделать акцент на проверке макета и результатах расклейки.

## Минимальная реализация

1. Добавить helper `spnWizardFlow.js`.
2. Разметить текущие секции шагами.
3. Добавить кнопки `Назад` и `Далее`.
4. В quick-режиме показывать только текущий шаг.
5. Продублировать выбор количества объявлений сразу после цели.
6. Сделать категории заготовок крупными карточками.

## Критерии готовности

1. Новый СПН может сделать макет 2 на А4 без объяснений.
2. Формат листа выбирается до шаблона.
3. Пустые макеты не показываются новичку по умолчанию.
4. Сложные настройки скрыты, но доступны.
5. Макеты с фото имеют нормальную компоновку изображения.
6. Перед печатью есть понятная проверка.

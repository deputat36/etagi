# Генератор расклейки

Модуль для подготовки печатных листов A4 из объявлений о недвижимости.

## Что входит

- `index.php` — конструктор листа расклейки: выбор объявлений, сетка, ориентация, отступы, печать.
- `editor.php` — визуальный редактор листовки с блоками: заголовок, текст, цена, контакты, изображение.
- `save_flyer.php` — сохранение макета в JSON.
- `data/ads.json` — временный источник объявлений до подключения MySQL.
- `data/flyers/` — сохраненные макеты.
- `assets/css/app.css` — стили интерфейса и печатного листа.
- `assets/js/app.js` — логика конструктора расклейки.
- `assets/js/editor.js` — логика визуального редактора.

## Запуск

На PHP-сервере открыть:

```text
/raskleyka/index.php
```

Редактор отдельной листовки:

```text
/raskleyka/editor.php
```

## Дальнейшая интеграция

Сейчас объявления берутся из JSON, чтобы модуль был автономным. Следующий шаг — заменить чтение `data/ads.json` на выборку из MySQL-таблицы объявлений или CRM.
